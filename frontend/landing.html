Security

Built to last, not to rug

Every security decision was made before the first line of code. Not patched after.

🔒
ReentrancyGuard
All state-changing functions protected. Checks-Effects-Interactions throughout.
✍️
EIP-712 Signatures
Typed structured data. Chain-specific domain. Replay attacks impossible.
⏱️
48h Timelock
All governance actions wait 48 hours. Community can react before execution.
📦
OpenZeppelin 5.x
Battle-tested libraries for tokens, access control, governance, and cryptography.
⚖️
Slashable Collateral
Solver misbehavior triggers stake seizure. Skin in the game by design.
🔑
Ownable2Step
Two-step ownership transfer. Typos can't lose the protocol to address(0).
🧊
7-day Cooldown
Unstake requests wait 7 days. Protects against pre-slash withdrawal attacks.
