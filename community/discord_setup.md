## REGRAS DO SERVIDOR (#📋-regras)

1. **Sem spam** — sem promoção de outros projetos sem permissão
2. **Sem previsões de preço** — "quando moon" não é discussão técnica
3. **Sem DMs não solicitados** — a equipe NUNCA vai te enviar DM pedindo seed phrase ou fundos. Se alguém fizer isso, é scam.
4. **Respeito** — comunidade técnica, linguagem profissional
5. **Português e inglês** são bem-vindos. Use #🌍-internacional para English.
6. **Contribuições são bem-vindas** — PRs no GitHub, ideias no Discord, bugs reportados

⚠️ **AVISO DE SEGURANÇA:** A equipe do Vortex nunca vai pedir sua chave privada, seed phrase, ou pedir para você enviar tokens para qualquer endereço. Qualquer mensagem assim é golpe.
